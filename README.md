# **背景故事**

在大厂当了3年程序员，但是总有一种危机感，这份危机感来源工作的内卷，残酷的年龄淘汰等等原因。同时我也希望能够有一份稳定不错的被动收入，能够支撑老了以后可以享受这个世界。

基于上述考虑，我毅然的裸辞，从北京回到成都创业

* 尝试过跨境电商
* 尝试自媒体带货  
* 尝试品牌加盟（餐饮，快消，教培...）
* 尝试过独立开发者
* 尝试过软件开发的招投标 
* 尝试开一家软件公司 
* 尝试找一级二级人脉合作  
* ....

尝试创业了3个月后，我又老老实实回去上班，为什么呢？

* 现金流：没有现金流，每天都在焦虑中，什么要快速看到结果，做什么都浅尝辄止
* 能力不够：创业看的不是做什么，而是看谁在做。深刻认识到自己的能力不足  
* 感受创业残酷：见了各种各样的创业者，成功的当然羡慕，失败的也让人唏嘘
* ...

我的感悟是什么呢？

* 还是要创业：上班挣的钱是就是线性的主动的（人劳动生钱），而创业可以实现财富的增值（钱生钱）
* 创业的理解：创业不一定开个公司，招一群人，做一个产品才是。不要局限创业的时间和空间，只要为用户创造了价值，一样可以实现财富增值 
* 信奉长期主义：现在的自媒体太多成功的人，教大家快速致富。但是从拿到结果的人来看，创业是一个艰苦孤独的路，要学会延迟满足。
* 业务引领技术：要发现业务后用最合适的技术去实现，而不是炫技式的塞到各种业务中

**为什么会有这个仓库？**

* 搜集创业的优质资料：创业的方法论，创业的行业洞察..
* 建立创业社群：建立一个创业者社群，共同成长，进步拿结果
* 自律：我希望可以倒逼自己做出一点输出

**这个仓库会更新的内容？**

* 搜集创业的优质课程(重点关注从到1的阶段方法论，能力)
* 搜集行业的深刻洞察，最新创业动态
* 共创好的想法和点子  
* 联创人招募

# 1.被动收入

[9种被动收入来源](https://www.xiaohongshu.com/explore/66f25c59000000001a02390b?note_flow_source=wechat&xsec_token=CBR8Wq5pA9ZSH9pWr2xQULsDFIuqqq0jyo87MKDwLmRAo=)

## 1.1什么是被动收入？

“被动收入” 常被我打上引号。严格来说，完全不努力就赚钱不可能。我所说的被动收入不直接与时间挂钩，且需很长时间起步。同时注意不存在快速致富方法，若在找快速致富方案应放弃。

## 1.2有哪些被动收入（9种）？

* **股票投资**
  
  * 概述：通过购买和持有股票，获得股息和资本增值
  
  * 适合人群：对金融市场有一定了解，愿意承担市场风险的人

* 创建自媒体
  
  * 概述：通过发布视频内容，吸引订阅者和观看次数，从广告收入中获利。
  
  * 适合人群：擅长内容创作和视频制作，能够持续提供有价值内容的人

* 创建博客
  
  * 概述：通过录制和发布音频内容，吸引听众和广告商，获得广告和赞助收入。
  
  * 适合人群：擅长交流，有独特观点和话题，并愿意与听众分享的人

* 做个买手
  
  * 概述：通过推广他人的产品或服务，获得佣金
  
  * 适合人群：有一定的网络影响力，善于内容营销和推广的人。

* 销售数字产品
  
  * 概述：制作并销售电子书、模板、插件等数字产品。
  
  * 适合人群：有专业技能或知识，能够制作高质量数字产品的人

* 在线课程，知识付费
  
  * 概述：将专业知识和技能制作成在线课程，供学员购买学习。
  
  * 适合人群：有教学能力，擅长系统化讲解和知识分享的人

* 创建付费会员或者社区模式
  
  * 概述：创建一个收费的会员社区，提供独家内容和服务。
  
  * 适合人群：有强大的社群运营能力，能够持续提供高价值内容的人。

* 创立公司（核心是业务自动化）
  
  * 概述：通过自动化工具和软件，简化和自动化业务流程，提高效率和收入。
  
  * 对自动化工具和技术有一定了解，愿意优化业务流程的人。

* 开发APP、网站或者软件产品
  
  * 开发并推出有用的应用程序或网站，通过订阅、广告或销售获利。
  
  * 有编程技能或能与开发团队合作，识别市场需求并提供解决方案的人。

## 1.3如何做被动收入？

我比较**关注股票投资**，**创立公司**，**开发APP网站软件**等以及**不动产**

* 股票投资：之前定投指数基金，以后只定投大盘指数（沪深300，恒生指数，标普500）

* 创立公司：字面意思

* 开发APP网站软间：以个人开发者开发一些有意思付费软件

* 不动产：投资商铺或写字楼

其他和自媒体相关的暂时不关注（自认为吃不了自媒体的饭）

# 5.好书推荐

1. **经济学**：
- **《经济学（第****19****版）》，保罗·萨缪尔森**：对各流派经济学有统筹性了解，是教材级读物，需通读并理解知识点。
- **《薛兆丰经济学讲义》，薛兆丰**：作为《经济学（第19版）》的教辅，帮助理解经济学知识。
- **《牛奶可乐经济学（****1 - 4****）》，罗伯特·弗兰克**：用理论联系实际，解释生活中现象背后的经济学原理。
- **《思考，快与慢》，丹尼尔·卡尼曼**：介绍行为经济学理念，帮助理解决策中的偏见与谬误。
- **《枪炮、病菌与钢铁》，贾雷德·戴蒙德**：从宏观角度理解世界发展及要素对经济和人类社会的影响。
- **《逃不开的经济周期》，拉斯·特维德**：建立经济周期的视角，了解经济的繁荣与萧条期。
3. **组织行为学**：
- **《组织行为学（第****18****版）》，斯蒂芬·罗宾斯**：从组织内部视角研究人在组织中的行为和态度，是教材级经典读物。
- **《李育辉组织行为学讲义》，李育辉**：作为补充读物，对组织行为学的前沿研究有解读。
5. **管理学**：
- **《管理学（第****13****版）》，斯蒂芬·罗宾斯**：从管理者角度思考如何改变和影响组织结构。
- **《卓有成效的管理者》，彼得·德鲁克**：帮助搭建德鲁克对管理认知的基本体系，强调管理者的关键要素。
7. **战略**：
- **《战略历程（第****2****版）》，亨利·明茨伯格**：归纳总结战略管理的“十大门派”，获得对战略的框架性理解。
- **《竞争战略》，迈克尔·波特**：深入理解竞争战略的原著，避免表面理解。
- **《平台战略》，陈威如**：讲述生态与生态之间的竞争及参与者与平台的合作模式。
9. **营销管理**：
- **《营销管理（第****16****版）》，菲利普·科特勒**：营销学的经典著作，需啃完建立营销框架。
- **《影响力》，罗伯特·西奥迪尼**：透彻讲解销售中的心理力量和“劝说”秘笈。
- **《定位》，艾·里斯，杰克·特劳特**：强调进入用户心智的定位观念。
- **《定价制胜》，赫尔曼·西蒙**：研究营销中的定价问题。
- **《乌合之众》，古斯塔夫·勒庞**：研究群体心理，是理解用户的重要角度。
- **《有闲阶级论》，索尔斯坦·凡勃伦**：研究上层阶级的消费特征和动机。
- **《自私的基因》，理查德·道金斯**：从基因角度解释人的需求背后的控制力量。
11. **金融学**：
- **《香帅金融学讲义》，香帅**：透彻讲解金融知识，包括金融架构和本质。
- **《基础会计（第****7****版）》，陈国辉**：涉及公司内部财务知识，是管理者的基本功。
- **《世界上最简单的会计书》，达雷尔·穆利斯**：会计入门读物，帮助理解基础会计。
- **《一本书读懂财报（全新修订版）》，肖星**：解读公司财报的三大财务报表，分析财务数据秘密。
13. **博弈论**：
- **《博弈论与经济行为》，冯·诺依曼**：研究合作博弈和决策行为的数学模型。
- **《妙趣横生博弈论》，阿维纳什****K****·迪克西特**：通过故事和例子展示博弈的艺术。
15. **中国商业史**：
- **《激荡三十年》，吴晓波**：记录改革开放30年企业变革的历史。
- **《激荡十年》，吴晓波**：可理解为记录激荡40年的历史。
- **《跌荡一百年》，吴晓波**：从不同角度展现中国商业史。
- **《浩荡两千年》，吴晓波**：从历史长河角度呈现中国商业史。
17. **创新**：
- **《创新者的窘境》，克莱顿·克里斯坦森**：提出“颠覆式创新”概念和相关理论。
- **《失控》，凯文·凯利**：用进化论逻辑解释创新，展示对时代判断的魅力。
19. **个人习惯**：
- **《高效能人士的七个习惯》，史蒂芬·柯维**：养成好的商业习惯，建议参加相关培训。
- **《金字塔原理》，芭芭拉·明托**：优化思维习惯，提升逻辑思考和表达能力。
- **《刘嘉概率论通识讲义》，刘嘉**：帮助认识概率论在商业世界中的重要性，管理概率以实现谋事和成事。
